Introduction


TrayClock Version 1.0 is a simple analogue clock designed as replacement for the standard windows clock in the tray notification area. It also has some more features than the windows clock, you can design the appearance of the clock and you can have up to 15 alarm events with different settings.

Startup


To start TrayClock just double click on the executable file. The program registers a new symbol in the tray notification area where you can left- and right-click on. With a left click on the tray symbol you can show the clock window (and wait for Auto Hide) or hide it with another click (if Auto Hide disabled). With a right click you can show the program menu.

Program Menu

About show up the about dialog of trayclock
Alarms show up the alarms dialog of trayclock
Settings show up the settings dialog of trayclock
Show Date check if you wish trayclock to show the current date
Round Window check if you wish to have a round window (shaped like the clock face)
Auto Hide check if you wish the clock window to hide after a specified amount of time (between 1 and 5 seconds)
Exit exit trayclock

Settings


Configuration Screen

In the configuration screen you can change settings for
Screen Positions ul - clock window is positioned in the upper left cornen of the screen
ur - clock window is positioned in the upper right corner of the screen
m - clock window is positioned in the middle of the screen
ll - clock window is positioned in the lower left corner of the screen
lr - clock window is positioned in the lower right corner of the screen
Colors background - select the clocks background color
foreground - select the clocks foreground color (clock face and hands, date)
special - select the clocks special color (second hand)
Auto Hide set duration the clock window is visible if auto hide is enabled
Window Size select clock window size
Opacity window transparency - makes the desktop and underlying windows shine through (100 % -> totally invisible)
clock shaped window - makes the background invisible
Refresh Time Refreshes the clocks time from the windows system time

Alarms


Alarm Screen

You can set up to 15 Alarm Events in TrayClock
To Add/Edit an alarm event click on Add.../Edit... This will show up the Add Alarm Screen.
Add Alarm Screen

Click on How To... to get a quick description on how to set alarm events
alarm mode select the alarm event mode
one shot - the alarm event will fire once and the be disabled
daily - alarm event will fire once per day
weekly - alarm event will fire every seven days from the specified date on
weekdays - alarm event will fire every weekday
weekend - alarm event will fire every day of the weekend
alarm time set the daytime the alarm event will fire
alarm date set the date the alarm event will (start) fire
only used if mode is one shot or weekly
alarm action play wave - on fire will play sound file specified in path
execute - on fire will execute file specified in path
shutdown - on fire will (totally force) windows shutdown
show messagebox - shows messagebox with text specified in text
active is the alarm active?


Every other button in the alarm screen should be self-descriptive
